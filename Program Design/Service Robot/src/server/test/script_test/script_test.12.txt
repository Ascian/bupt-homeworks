
   Step welcome  
     Speak $name + "您好，请问有什么可以帮您?"
     Listen 5, 20
     Branch "投诉", complainProc
     Branch "账单", billProc
     Silence silenceProc

     Default  #缺少步骤名标志符

    Step complainProc
     Speak "您的意见是我们改进工作的动力，请问您还有什么补充?"
     Listen 5, 50
     Silence thanks
     Default thanks
    Step thanks
     Speak "感谢您的来电，再见"
     Exit
    Step billProc
     Speak "您的本月账单是" + $amount + "元，感谢您的来电，再见" 
     Exit
    Step silenceProc
     Speak "听不清，请您大声一点可以吗"
     Branch "投诉", complainProc
     Branch "账单", billProc
     Silence silenceProc
     Default defaultProc
    Step defaultProc
     Speak "无法理解您的意思，请重新说一遍可以吗"
     Branch "投诉", complainProc
     Branch "账单", billProc
     Silence silenceProc
     Default defaultProc
